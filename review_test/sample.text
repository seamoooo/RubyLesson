課題内容
Web APIを利用して、あるユーザーIDをあたえたときのユーザー名および登録金融機関一覧と、
登録金融機関の残高(balance)を出力するプログラムを作成してください。

実施時間
70分

実行環境
ruby 3.0.2
httpclient 2.8.3

動作確認方法
gemのinstall
gem install httpclient -v 2.8.3

上記のファイルを同じ階層のディレクトリに配置して、下記コマンドを実行すると
下記の返り値が取得できる
ruby financial_institutions_info_output.rb

ユーザーIDを入力してください
1
-----------------------------
ご登録中のユーザー名
Alice
-----------------------------
各登録金融機関名及び、各登録金融機関の残高
-----------------------------
金融機関名: A銀行
残高: 20000
-----------------------------
金融機関名: C信用金庫
残高: 120000
-----------------------------
金融機関名: E銀行
残高: 5000
-----------------------------

実装について考慮した点
APIコール数を減らすために、/accounts/ account_id ではなく、/users/user_id/accountsでuser_idに紐づく金融機関を取得
実装をAPIデータの取得(repository)、データの加工(service)、
実行ファイルへのデータの受け渡し(presenter)、出力処理の実装(output)の4つの層に分割
status codeが200番以外はエラーが発生するようにする
